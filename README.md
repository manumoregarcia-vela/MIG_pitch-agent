# MIG_pitch-agent
Sistema de agentes para la revisión y creación de pitch para publishers e inversores

## Objetivo
Generar automáticamente decks de 5-8 slides optimizados para publishers e inversores.

## Inputs
- Decks
- Web
- Steam
- Redes sociales
- Métricas

## Outputs
- structured_summary.json
- pitch_strategy.md
- slides_outline.json
- draft_deck.md
- pptx
